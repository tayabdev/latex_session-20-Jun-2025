  • Double-click the .pkg file
  • Follow installation wizard
  • Enter admin password when prompted

üîçStep 2: Verify Your LaTeX Installation

Open Terminal (found in Applications ‚Üí Utilities) and run these commands to check your setup:

Check if pdflatex is installed:

which pdflatex

Expected output: /Library/TeX/texbin/pdflatex

Check LaTeX version:

pdflatex --version

Expected output: Should show "pdfTeX 3.141592653" and "TeX Live 2025"

Check package manager:

sudo tlmgr --version

Expected output: Should show "tlmgr revision" and installation path

Verify TeX directory:

ls /usr/local/texlive/

Expected output: Should show "2025" and "texmf-local" folders

‚úÖ Verification Checklist:

pdflatex command found in /Library/TeX/texbin/
Version shows TeX Live 2025
tlmgr package manager working
TeX Live directory exists
TeX app visible in Applications folder

üõ†Ô∏èStep 3: Create Your First LaTeX Document

1
Create a new file using Terminal:
nano hello.tex

This opens a simple text editor in Terminal.

2
Copy and paste this LaTeX code:
\documentclass{article} \usepackage[utf8]{inputenc} \title{My First LaTeX Document} \author{Your Name} \date{\today} \begin{document} \maketitle \section{Introduction} Hello, LaTeX! This is my first document created on Mac with Apple Silicon. \section{Mathematics} LaTeX is great for math: $E = mc^2$ \section{Lists} Here's a simple list: \begin{itemize} \item First item \item Second item \item Third item \end{itemize} \end{document}
3
Save and exit the editor:
  • Press Ctrl + O to save
  • Press Enter to confirm filename
  • Press Ctrl + X to exit

‚ö°Step 4: Compile Your Document

1
Compile the LaTeX file to PDF:
pdflatex hello.tex
Success indicators:
  • No error messages appear
  • Output ends with: "Output written on hello.pdf (X pages, Y bytes)"
  • Files created: hello.pdf, hello.aux, hello.log
2
Open your created PDF:
open hello.pdf

This opens the PDF in Preview or your default PDF viewer.
