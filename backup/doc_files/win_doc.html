üíªStep 2: Install MiKTeX

1
Run the installer:
  • Double-click the downloaded .exe file
  • Click "Yes" if Windows asks for permission
2
Installation options:
  • Install for: "All users" (recommended if you have admin rights)
  • Installation directory: Keep default (C:\Program Files\MiKTeX)
  • Install missing packages: Choose "Yes" (install on-the-fly)
  • Paper size: A4 or Letter (your preference)
3
Complete installation:
  • Click "Next" through all prompts
  • Installation takes 5-10 minutes
  • Don't close Command Prompt windows during installation

What Gets Installed:

MiKTeX LaTeX distribution
TeXworks editor (basic GUI editor)
MiKTeX Console (package manager)
Command line tools (pdflatex, etc.)

üß™Step 4: Test Your Setup

1
Create a test LaTeX file:

Create a new file called hello.tex with this content:

\documentclass{article} \usepackage[utf8]{inputenc} \title{My First LaTeX Document on Windows} \author{Your Name} \date{\today} \begin{document} \maketitle \section{Introduction} Hello, world! This is my first LaTeX document on Windows. \section{Mathematics} LaTeX handles math beautifully: $E = mc^2$ \section{Lists} Here's what I can do with LaTeX: \begin{itemize} \item Create professional documents \item Write mathematical equations \item Generate bibliographies \item Make presentations \end{itemize} \end{document}
2
Compile using Command Prompt/PowerShell:
# Open Command Prompt (Win + R, type cmd) # Navigate to your file location cd C:\Users\YourUsername\Desktop # Compile the LaTeX file pdflatex hello.tex
3
Check the results:
# Open the generated PDF start hello.pdf # Or list files to see what was created dir *.pdf
Success indicators:
  • No error messages during compilation
  • Files created: hello.pdf, hello.aux, hello.log
  • PDF opens and displays formatted content

üö® Troubleshooting Tips:

  • Command not found: Restart Command Prompt or reboot Windows
  • Package errors: Install missing packages via MiKTeX Console
  • Permission errors: Run Command Prompt as Administrator
  • Path issues: MiKTeX should auto-configure PATH during installation
  • Editor won't compile: Check if MiKTeX path is configured in editor settings
