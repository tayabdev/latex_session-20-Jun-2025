Creates definition-style lists with custom labels

üíª7. Code and Algorithms

Code Listings
Displaying source code with syntax highlighting
\usepackage{listings} \usepackage{xcolor} \lstset{ language=Python, basicstyle=\ttfamily\small, keywordstyle=\color{blue}, commentstyle=\color{green}, stringstyle=\color{red}, numbers=left, numberstyle=\tiny, frame=single } \begin{lstlisting}[caption=Linear Search Algorithm] def linear_search(arr, target): for i in range(len(arr)): if arr[i] == target: return i return -1 \end{lstlisting}
Formatted code blocks with syntax highlighting and line numbers
Algorithm Pseudocode
Formal algorithm presentation
\usepackage{algorithm} \usepackage{algpseudocode} \begin{algorithm} \caption{Binary Search} \begin{algorithmic}[1] \Procedure{BinarySearch}{$arr, target$} \State $left \gets 0$ \State $right \gets length(arr) - 1$ \While{$left \leq right$} \State $mid \gets \lfloor(left + right) / 2\rfloor$ \If{$arr[mid] = target$} \State \Return $mid$ \ElsIf{$arr[mid] < target$} \State $left \gets mid + 1$ \Else \State $right \gets mid - 1$ \EndIf \EndWhile \State \Return $-1$ \EndProcedure \end{algorithmic} \end{algorithm}
Professional algorithm presentation with formal notation

üéØ8. Special Environments

Theorems and Proofs
Mathematical theorems and formal proofs
\usepackage{amsthm} \newtheorem{theorem}{Theorem} \newtheorem{lemma}{Lemma} \newtheorem{definition}{Definition} \begin{theorem} For any real number $x$, $e^x \geq 1 + x$. \end{theorem} \begin{proof} Define $f(x) = e^x - 1 - x$. Then $f'(x) = e^x - 1$ and $f''(x) = e^x > 0$. Since $f'(0) = 0$ and $f''(x) > 0$, $f(x)$ has a global minimum at $x = 0$. Therefore, $f(x) \geq f(0) = 0$. \end{proof}
Creates numbered theorems with formal proof environments
